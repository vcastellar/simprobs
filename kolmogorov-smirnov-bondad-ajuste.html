Calculadora de Kolmogorov–Smirnov (bondad de ajuste)

Contrasta si una muestra continua sigue una distribución teórica.

Volver a contrastes

Descripción

Test no paramétrico basado en la distancia máxima entre CDF empírica y teórica.

Hipótesis y estadístico

\(H_0\): la muestra sigue la distribución teórica; \(H_1\): no la sigue.

\(D=\sup_x |F_n(x)-F_0(x)|\).

Contraste rápido

No depende de clases, pero si parámetros se estiman con la muestra hay que usar ajustes (Lilliefors).

¿Qué calcula esta herramienta?

Esta página está preparada con el mismo esquema que el resto de contrastes de ProbLab e incluye una sección de calculadora para introducir datos y estructurar el análisis.

Cómo interpretar el resultado

Si p-valor < α, se rechaza \(H_0\). Si p-valor ≥ α, no hay evidencia suficiente para rechazar \(H_0\).

Preguntas frecuentes

  • ¿Es paramétrico? No, es no paramétrico.
  • ¿Se puede usar para normalidad? Sí, aunque con ajuste de Lilliefors si parámetros se estiman.
