Descripción

Contrastes para analizar si la variabilidad de un proceso coincide con una hipótesis teórica o si difiere entre dos grupos independientes.

Hipótesis y estadístico

\(H_0: \sigma^2=\sigma_0^2\) o \(H_0: \sigma_1^2=\sigma_2^2\).

\(\chi^2= rac{(n-1)s^2}{\sigma_0^2}\) y \(F= rac{s_1^2}{s_2^2}\).

Cómo interpretar el resultado

Si p-valor < α, se rechaza \(H_0\). Si p-valor ≥ α, no hay evidencia suficiente para rechazar \(H_0\).

Preguntas frecuentes

  • ¿Cuándo usar chi-cuadrado o F? Chi-cuadrado para una varianza; F para comparación de dos varianzas.
  • ¿Es sensible a no normalidad? Sí, especialmente con tamaños pequeños.
