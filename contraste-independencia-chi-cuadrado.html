Descripción

Contraste para verificar si dos variables categóricas son independientes o muestran asociación estadística.

Hipótesis y estadístico

\(H_0\): independencia entre variables; \(H_1\): asociación.

\(\chi^2=\sum rac{(O-E)^2}{E}\).

Cómo interpretar el resultado

Si p-valor < α, se rechaza \(H_0\). Si p-valor ≥ α, no hay evidencia suficiente para rechazar \(H_0\).
